# This project will be a personal AI operating system designed to manage knowledge, memory and my workflows. The goal is to create a system that can automate tasks, has all my information across my chats, gmails, PDFs, git repositories, and calenders, has a lot of context about my situation, more than a regular LLM like ChatGPT or Claude would have on any given topic.

# The system will be designed to be modular and extensible, allowing for the integration of various AI models and tools. It will also have a user-friendly interface that allows me to easily access and manage my information and workflows.

# I will be using Next.js and React for the frontend. And the rest will be updated as this project progresses. I will also be making content about my progress.
